World of Warcraft Themed Dungeon App

Project information

  • Demonstration of Technology Proficiency using C#
  • Project date: July 2023
  • Link to Repository

World of Warcraft Themed Dungeon App - User is given menu options to choose a character and a weapon. In randomly generated rooms, user fights monsters in a queue. User is continuously offered a menu to attack, view player stats, monster stats, and/or exit the game and will continue to play until they quit the game.

Designed by Melissa Balino using BootstrapMade
